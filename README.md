# Pokedex Web (C# ASP.NET Core)

A web-only Pokedex application built with C# and ASP.NET Core. Browse, search, and view Pokémon details in a modern web UI.

## Features

- Browse Pokémon list
- Search Pokémon by name
- View Pokémon details

## Getting Started

1. Ensure you have the .NET SDK installed (https://dotnet.microsoft.com/download)
2. Build and run the project:
   ```sh
   dotnet run --project Pokedex
   ```
3. Open your browser at http://localhost:5000

## Notes

- This project is web-only (no mobile/desktop components).
- Pokémon data is fetched from a public API or static file (to be implemented).
